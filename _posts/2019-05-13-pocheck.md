# 얼굴 인식 출석 시스템 <br>

사용자의 얼굴에서 pre-trained facenet model을 이용해 feature 추출<br>
얼굴 식별을 위해 svm과 joint-bayesian기법 사용<br>
Django를 이용해 웹 서비스 구축<br>

### 사용 스택 <br>

`Python` `tensorflow` `Django`

<br>

### 얼굴 특징 추출 모델
```
facenet 
```

### 안면 인식 모델
```
Joint-Bayesian
```
